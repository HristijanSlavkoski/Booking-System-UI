VR Escape Room Booking

WordPress Integration Guide

This page demonstrates how to embed the booking system into your WordPress site. The iframe seamlessly integrates with your dark theme.

URL Parameters

Pass data to the booking system via URL parameters:

Basic Booking (No Game Selected)

<iframe src="https://yourangularapp.com/booking" width="100%" height="900px"></iframe>

Pre-select a Game

<iframe src="https://yourangularapp.com/booking?gameId=depths-of-osiris" width="100%" height="900px"></iframe>

Pre-select Game + Players

<iframe src="https://yourangularapp.com/booking?gameId=depths-of-osiris&players=4" width="100%" height="900px"></iframe>

Available Parameters

• gameId - The game identifier (e.g., "depths-of-osiris", "dragon-tower")
• players - Suggested number of players (e.g., 2, 4, 6)

WordPress Implementation

On your "Our Rooms" page, each game card should link to the booking page with the game's ID:

<a href="/appointment?game=depths-of-osiris" class="book-now-btn">BOOK NOW</a>

Then on the appointment page, embed the iframe with the game parameter from the URL query string.
